Terms used

Gloss
A definition of a word in WordNet.
Root node
A virtual node connecting all the hierarchies together. The nouns and verbs in WordNet are arranged in hierarchies. There are nine noun hierarchies and several hundred verb hierarchies. By turning this option on, one unique root node joins all the noun hierarchies together, and another node joins all the verb hierarchies together.
Synset
Short for synonym set. A non-empty set of synonyms. E.g., {dog#n#1, domestic_dog#n#1, canis_familiaris#n#1}
